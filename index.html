Site de testes

Esta pagina é um projeto de testes desenvolvido com o intuito de praticar HTML e CSS. O objetivo é criar uma página simples com estilização e funcionalidades básicas, como forma de aprendizado, para posteriormente aplicar as capacidades desenvolvidas em projetos mais complexos.

Artigo

Aqui será posicionado um artigo de exemplo, onde o conteúdo pode ser extenso o suficiente para testar a rolagem dentro do artigo. O objetivo é verificar se a estrutura da página suporta um layout flexível e se o conteúdo é exibido corretamente, mesmo quando excede a altura disponível. Este artigo serve como um teste para garantir que a formatação e a apresentação do texto sejam mantidas, independentemente do tamanho do conteúdo.

Abaixo, pode-se observar uma imagem meramente ilustrativa, que demonstra a responsividade da página: Para resumir, eu estou desenvolvendo este projeto como uma base para aprender e práticar HTML, CSS e, posteriormente, Javascript. Planejo utilizar estas capacidades em meu atual projeto principal, o "Novo ficheiro", de forma a criar uma página web funcional e interativa, que possa ser utilizada como interface gráfica para facilitar o uso da aplicação, que atualmente requer que o usuário faça os requests diretamente, com o uso de programas como Insomnia ou Postman. pretendo seguir a estrutura básica de uma página web, com um header, footer, e um corpo principal, onde o conteúdo será exibido.
